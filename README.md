# ph-events
A simple Java eventing library

# Maven usage
Add the following to your pom.xml to use this artifact:
```xml
<dependency>
  <groupId>com.helger</groupId>
  <artifactId>ph-events</artifactId>
  <version>5.0.0</version>
</dependency>
```
